E2E Application Parameters


E2E : End to End entanglement generation between any two nodes
Sender : name of node to establish entanglement from
Receiver : name of node to establish entanglement with
startTime : simulation start time of entanglement
Size : number of entangled memory pairs to create
Priority : Priority of the request
targetFidelity : desired fidelity of entanglement
timeout : From startTime the time window within which entanglements need to be generated
